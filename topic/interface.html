Interface:-

create interface we use "interface" keyword. interface is 0 implemented class. interface contains all abstract method. all abstract method have a public access specifier by default. interface can have properties ,Automatically this properties are final and public by default. interface can have used to achieve polymorphism multiple inheritance and loose coupling. Save program car.java
interface car
{
int max=150;
void wheel();
void color();
void speed();
}
************
save program Honda.java

class Honda implements car
{
public void wheel()
{
System.out.println("4 wheel");
}
public void color()
{
System.out.println("black");
}
public void speed()
{
System.out.println("40");
}
public static void main(String args[])
{
Honda h=new Honda();
h.wheel();
h.color();
h.speed();
//h.max=100; we cant modified value into variable due to final
}
}
