Control Structure

1) if_:-

  • -if the condition requires a boolean true value.
  • -we can use relational operators and logical operator to check a condition.
  • -we can write two more statements into if condition.
  • -we can write a condition without using else. -else past execute only will become false.

Example :-

save program with class name if_demo.java
class if_demo
{
public static void main(String args[])
{
if(true)
{
System.out.println("True block printed false block is terminated");
}
else
{
System.out.println("False block");
}
}
}

2)if_else_if :-

  • -This condition also known as ladder _if condition.
  • -the compiler will check every condition from top to bottom.
  • -if any condition satisfied inner statement will be executed, the rest part will be ignored by the compiler.
  • -Else part will be executed only condition will not satisfy by any if block.

    Example:-

    class if_else_if
    {
    public static void main(String args[])
    {
    int time=10;
    if(time>=00 && time<=12)
    {
    System.out.println("Good Morning");
    }
    elseif(time>=12 && time<=17)
    {
    System.out.println("Good Afternon");
    }
    elseif(time>17 && time<24)
    {
    System.out.println("Good Evening");
    }
    }
    }

    4)Switch:-

  • -Switch condition can be considered's false the other.
  • -we can check only one variable at a time using a switch condition.
  • -There can be inner switch into switch condition
  • -we can check values by using the "case" keyword.
  • Note:-
  • -when the case satisfied, the rest else will be executed.
  • -To overcome this problem we can use the "Break" keyword.
  • -if the case not executed default part will be executed, the default block can be developed using the "default" keyword

    Example:-

    class switch_eg
    {
    public static void main(String args[])
    {
    int a=2;
    switch(a)
    {
    case 1:
    {
    System.out.println("Coffie");
    break;
    }
    case 2:
    {
    System.out.println("Tea");
    break;
    }
    case 3:
    {
    System.out.println("Green Tea");
    break;
    }
    case 4:
    {
    System.out.println("Black Coffie");
    break;
    }
    default:
    {
    System.out.println("Out of number"+a);
    break;
    }
    };
    }
    }
