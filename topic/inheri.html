Inheritance:-

inheritance is a concept where one class acquires property of another class. this concept is used to when, code reusability required .

single inheritance

In single inheritance one parent one child class. class who acquire property of another class considered as a child class. a child class can be known as derived class . A Class who provides a properties to the others class is known as parent class. parent class can be known as base and super class

multilevel inheritance

in a multilevel inheritance class A acquire property from another class known as a child class. this class can be considered as a parent class. if this class extend by child class. A class which not extend the property of another class is known as parent as a superclass. object class can be considered as super class.

heretical inheritance

In this two or more class can extend a single class. hence one part can have two or more child class or vice versa.
