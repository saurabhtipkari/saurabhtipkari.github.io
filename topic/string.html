String:-

string is a class which is located into "java lang.package". string object is used to store string data ,which stores character as array basic. string always start from 0 index and with length-1. string class considered as immutable class. immutable means we cannot modify string object letter on. string class follow string constant pool concept. if string constant pool contains a given hashcode ,string object will refer refer to old object. it is not available hash code will store into string constant pool and reference will be generated. when object creation done through new hashcode will be generated.

Stringbuffer

Stringbuffer class is mutable class which can be modify later on. stringbuffer class does not have concat() method for concatenation. to concatenate string buffer provider appends() method to concatenation. this append() is synchronised hence stringbuffer considered as a thread safe.
