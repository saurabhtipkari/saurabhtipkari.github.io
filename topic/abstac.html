Abstraction :-

abstraction is a process of implementation hiding. one class can have abstract and non abstract method as well. if any class contains single abstraction ,automatically non implemented Method can be considered as a abstract. class can be concrete as 0 to 100% implemented class. once a class declared as an abstract object creation cannot be possible. we can only call a method directly or overridden hence implementation will be hidden.

Example:-

abstract class opration
{
public void add(int a,int b)
{
System.out.println(a+b);
}
public void sub(int a,int b)
{
System.out.println(a-b);
}
public void mul(int a,int b)
{
System.out.println(a*b);
}
}
class cal extends opration
{
public void div(int a,int b)
{
System.out.println(a/b);
}
public void mul(int a,int b)
{
System.out.println("My method "+(a/b));
}
}

class abstration
{
public static void main(String args[])
{
cal c=new cal();
c.add(20,60);
c.mul(3,2);
}
}
