Encapsulation :-

Encapsulation is a data binding process in data binding process. In binding process data hide by making property as a private to manipulate thus properties we can write setter and getter method. to display a class we can overside tostring method of object class . encapsulated class is 100% conceder as concrete class.

Example


class product_encapsulation //run the program of shop.java first
{
private int id;
private String name;
private String des;
private double price;
//setter
public void setid(int id)
{
this.id=id;
}
public void setname(String name)
{
this.name=name;
}
public void setdes(String des)
{
this.des=des;
}
public void setprice(double price)
{
this.price=price;
}
//getter
public int getid()
{
return id;
}
public String getname()
{
return name;
}
public String getdes()
{
return des;
}
public double getprice()
{
return price;
}
public String toString()
{
return "id = " + id +
",name = " + name +
",price= " + price +
",des= " + des +
"--" + printMe("who", "ABC");
}
public String printMe(String what, String value){
return what + " -- " + value;
}
}
